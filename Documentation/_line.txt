NAME 
      _line -- Display a line of a given length 
  
SYNOPSIS 
      _line [length] {char} 
  
OPTIONS 
  -h      Help page 
  length  The length of the line  
  char    The charactor used to draw the line [Default: #]  
  
DESCRIPTION 
  Displaying separator lines to improve the layout  
  
EXAMPLES 
      CALL _line 10 
 will display 
      ########## 
  
      CALL _line 5 X 
 will display 
      XXXXX
