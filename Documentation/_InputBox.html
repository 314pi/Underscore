NAME %$NAME% -- %$DESCRIPTION% SYNOPSIS _InputBox [Window title] [Message] [Default value] OPTIONS -h Help page DESCRIPTION Prompting user for simple text input using input box EXAMPLES SET _INPUT= SET _TITLE=window title SET _MSG=Hello world.{CrLf}Please enter something for me: SET _CMD=CALL %$NAME% "%_TITLE%" "%_MSG%" "default" FOR /F "tokens=* delims=" %%a IN ('CALL %_CMD%') DO SET _VAR=%%a ECHO You entered [%_VAR%] The following exit values are returned: 0 Any matches were found. 1 No matches found.
