NAME

      unittest -- Unittest for Underscore 
  

SYNOPSIS

      unittest 
      unittest [function] 
  

OPTIONS

  -h      Help page 
 

DESCRIPTION

  Runs a test suite for the function. 
  If a function is given as argument, this specific function is tested. 
  If no argument is given, the entire suite is tested.
