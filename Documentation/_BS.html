NAME %$NAME% -- %$DESCRIPTION% SYNOPSIS %$NAME% OPTIONS -h Help page DESCRIPTION Set an environment variable to hold a backspace character. Used in text manipulation on screen EXAMPLES CALL %$NAME% SET /P "_=a-!_BS!z"nul ) ECHO Done! GOTO :EOF :LOOP SET /A _=%~1 %% 4 IF "0" == "!_!" SET /P "_=^|!_BS!">NUL IF "1" == "!_!" SET /P "_=/!_BS!">NUL IF "2" == "!_!" SET /P "_=-!_BS!">NUL IF "3" == "!_!" SET /P "_=\!_BS!">NUL GOTO :EOF ENVIRONMENT _BS Holds the backspace character REQUIRES _Debug.cmd Setting up debug environment for batch scripts _GetOpt.cmd Parse command line options and create environment vars SEE ALSO _cr.cmd Carriage return _LF.cmd Line Feed REFERENCE Author: Ed Dyreen URL: http://www.dostips.com/forum/viewtopic.php?f=3&t=2124 SOURCE C:\Users\Erik\Documents\GitHub\Underscore\_BS.cmd (c)2015 Erik Bachmann, ClicketyClick.dk [ErikBachmann@ClicketyClick.dk]
