NAME 
      driveTypes -- List local drives available by type 
  
SYNOPSIS 
      driveTypes [VAR] 
  
OPTIONS 
  -h      Help page 
  
DESCRIPTION 
  List all drives alphabetically and determine type 
  
EXAMPLES 
      driveTypes 
      drlist=[C:\ D:\ E:\ ] 
      C: - Fixed Drive 
      D: - CD-ROM Drive 
      E: - Remote/Network Drive 
  
 
SOURCE 
  C:\_\driveTypes.cmd
