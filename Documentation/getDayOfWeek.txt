NAME 
      getDayOfWeek -- Get day of week for a given date 
  
SYNOPSIS 
      getDayOfWeek [VAR] [YYYY-MM-DD] 
  
OPTIONS 
  -h      Help page 
 
DESCRIPTION 
  Get day number in week: Man - Sun 0 - 6  
  
EXAMPLES 
  
      getDayOfWeek _dow 2015-03-31 
      set _dow 
      _dow=1
