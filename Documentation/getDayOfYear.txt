NAME 
      getDayOfYear -- Get day number in year 
  
SYNOPSIS 
      getDayOfYear [VAR] [Date YYYY-MM-DD] 
  
OPTIONS 
  -h      Help page 
 
DESCRIPTION 
  Get day number in year for a given date 
  
EXAMPLES 
 
        getDayOfYear _doy 2015-03-30 
        SET _doy=89
