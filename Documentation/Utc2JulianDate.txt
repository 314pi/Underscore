NAME 
      Utc2JulianDate -- Convert UTC date [YYYY-MM-MM] to Julian date [Numeric] 
  
SYNOPSIS 
  Utc2JulianDate {function} {PIPE} ALL 
 
OPTIONS 
  -h      Help page 
 
DESCRIPTION 
  Converting af human readable UTC date YYYY-MM-DD to a  
  numeric Julian date 
 
EXAMPLES 
  CALL utc2JulianDate #JD '2010-01-20' 
  SET #JD 
  #JD=2455217 
 
SEE ALSO 
  JulianDate2Utc
