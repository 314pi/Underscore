NAME

      tail -- Print the last n lines of a file to STDOUT 
  

SYNOPSIS

      tail [File] [n] 
  

OPTIONS

  -h      Help page 
 n   Number of lines to print, default=10  
  

DESCRIPTION

  A function like MORE, but displays only the last given number of lines
