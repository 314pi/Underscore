NAME 
      wget -- Search file for pattern '@(#)' and print remainder of string 
  
SYNOPSIS 
      wget [URL] [Output file] 
  
OPTIONS 
  -h      Help page 
  
DESCRIPTION 
  Used for downloading files from the internet 
  using a command line batch tool. 
  
EXAMPLES 
  wget 'http://clicketyclick.dk/development/dos/_/current' '{PCT}TEMP{PCT}\current.txt'
