NAME 
      _Config -- Global configuration file 
  
SYNOPSIS 
      CALL _Config 
    
OPTIONS 
  -h      Help page 
 
DESCRIPTION 
  Setting up global vars needed by most scripts like logging, trace etc.  
  
ENVIRONMENT 
  _LogFile    Location of log file 
  _TraceFile  Location of trace file 
  _LOG_       Log command 
  _TRACE_     Trace command 
  _FINAL_     Footer
