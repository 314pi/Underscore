NAME 
      _parentDir -- Get parent directory to a given directory 
  
SYNOPSIS 
      _parentDir [VAR] {dir} 
  
OPTIONS 
  -h      Help page 
  VAR     Name of variable to return value in 
  DIR     A given directory 
  
DESCRIPTION 
  
  
EXAMPLES 
      CALL _parentDir XX 'C:\Windows\system32\' 
      SET XX 
  Will display: 
      XX=C:\Windows
