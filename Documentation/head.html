NAME

      head -- Print the first n lines of a file to STDOUT 
  

SYNOPSIS

  head [File] [n] 
  
  n = Number of lines to print, default=10 
  

OPTIONS

  -h      Help page 
 

DESCRIPTION

  A function like MORE, but stops output after a given number of lines  
  

EXIT STATUS

  0   OK 
  1   File not found
