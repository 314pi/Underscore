NAME

      which -- Searching for an executable on the path (or other path-like string if necessary) 
  

SYNOPSIS

  which filename 
  

OPTIONS

  -h      Help page 
 

DESCRIPTION

  DEPRICATED - Use: Where.exe [If available] 
  Used for identifying the location of an excetutable 
  

EXAMPLES

      CALL which reg 
  Should echo 
      C:\Windows\System32\reg.exe
